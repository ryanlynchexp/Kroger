# Kroger Value Realization — Experis micro-site

Experis-branded click-through micro-site communicating the Kroger cost-savings plan.

- `index.html` — the micro-site (open directly or serve statically)
- `assets/` — Experis & Kroger logos, icons
- `downloads/` — Executive one-pager (PDF), full deck (PDF), savings model (XLSX)

## Hosting on GitHub Pages
This repo is structured for GitHub Pages: `index.html` is at the root.
Enable Pages under **Settings → Pages → Branch: main / root**.
